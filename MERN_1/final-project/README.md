# 🛒 Final Project — E-Commerce Inventory Auditor

This project is a browser-based **E-Commerce Inventory Auditor** built using:

- JavaScript (Prototypes + Closures)
- Async JavaScript (Fetch API)
- DOM Manipulation
- Debounce & Throttle
- ES6+ (Map, Set, Reduce)
- TypeScript (Generics, Utility Types, Abstract Classes)
- Vite (Build Tool)

This project combines everything learned in MERN JavaScript & TypeScript training.
