# Task 05 — Dynamic Form Validator SPA (Pure DOM)

## Run
Open `index.html` in the browser (or use VS Code Live Server).

## Features
- Pure DOM SPA (single page)
- Querying via `querySelector` / `querySelectorAll`
- DOM manipulation: create/append, attributes, classList, innerHTML
- Add/Edit/Delete tasks
- **Event delegation** on task list (single click handler on parent)
- Live debounced validation (email + pattern)
- Throttled resize handler for layout changes
- `preventDefault()` on form submit
- Persistence using `localStorage` with JSON serialization
